# Динамічний масив у Python: короткий опис та результати

* Реалізовано клас DynamicArray, що працює через ctypes і автоматично збільшує capacity при додаванні елементів.

* Передбачено три стратегії зростання:

* Double – подвоєння ємності

* Fixed – множник 1.4

* Dynamic – 1.4 / log₂(size+2)

* Підтримуються доступ get/set, ітерація та конвертація в список.

# Ефективність стратегій

* Double – найшвидша, але залишає більше невикористаної пам’яті

* Fixed – потребує більше resize, але економніша

* Dynamic – найповільніша через часті розширення

* Додавання масштабується краще при більшій початковій capacity та агресивному рості.
